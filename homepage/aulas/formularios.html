Formulários

Formulários são feitos utilizando a tag 'form'.
Algumas definições:

  • input: Permite interação com o usuário;
  • input type="": Define o tipo (ex.text);
  • input type="button" : Define um botão;
  • input type="button" value="" : Value define o nome do botão;
  • input name="": Permite a gravação dos dados a partir de uma linguagem de programação.
  • input type="submit": Enviar os dados.

Tabelas

Para fazer uma tabela em html, utilizamos as tags 'table', 'tr'(linha) e 'td'(coluna).

Formatando uma tabela

Utilize as seguintes definições:

  • border="" : Define a borda;
  • tag th : Define um cabeçalho;
  • width="": Define a largura;
  • colspan="" : Agrupa colunas
  • rowspan="" : Agrupa Linhas
